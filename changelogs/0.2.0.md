# v0.2.0 - 2026-03-18

## 版本概览

本版本完成个人版架构 Phase 1 的核心落地：

1. 引入会话实体与会话管理接口。
2. 文档与聊天记录实现会话级隔离。
3. 前端支持真实会话的新建、切换、删除。
4. 聊天默认可直接对话，有索引时再进入 RAG 链路。

## 新增内容

1. 新增会话模型、Schema、Service、路由（`POST/GET/DELETE /api/v1/conversations`）。
2. 新增会话隔离相关测试用例。
3. 新增产品与架构设计文档：`design/personal-edition-architecture.md`。

## 主要变更

1. `conversation_id` 贯穿 chat/document/retrieval/history 请求与响应结构。
2. 文档导入、文档列表、索引构建、检索查询支持会话范围。
3. 聊天历史支持按会话写入与查询过滤。
4. 前端“新会话”从仅清屏改为后端真实会话管理。
5. 项目版本号升级到 `0.2.0`。

## 兼容性说明

1. 对已有 SQLite 数据文件增加轻量启动迁移：
   1. `documents.conversation_id`
   2. `chat_history.conversation_id`
2. 主要接口保持向后兼容：新增字段尽量采用可选参数。

2. 前端脚本语法检查通过（`node --check app/web/app.js`）。
