# v0.5.0 - 2026-03-19

## 版本概览

本版本完成了会话与消息交互的 GPT 风格升级：

1. 会话操作整合为 `...` 菜单，并新增顶置聊天能力。
2. 用户消息与助手消息都支持外置小图标操作。
3. 助手消息支持复制与切换模型重生成。

## 新增内容

1. 新增会话顶置接口：`PATCH /api/v1/conversations/{conversation_id}/pin`。
2. 新增会话置顶字段：`is_pinned`、`pinned_at`。
3. 新增会话置顶排序规则：置顶优先，再按时间排序。
4. 前端新增会话 `...` 操作菜单（重命名/顶置聊天/删除）。

## 主要变更

1. 会话列表操作从分散按钮改为菜单式入口，交互更统一。
2. 聊天消息操作改为小图标形式，减少视觉噪音。
3. 助手消息增加“复制”“重生成”图标操作，重生成支持模型切换。
4. 启动时自动兼容 SQLite 旧表结构（补齐会话置顶相关字段）。

## 兼容性说明

1. 本次新增字段向后兼容，不影响既有会话数据读取。
2. 旧会话默认未置顶，行为与旧版本一致。
