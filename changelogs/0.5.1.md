# v0.5.1 - 2026-03-19

## 版本概览

本版本是前端 UI 细节优化版本：

1. 将用户消息操作图标移动到用户气泡下方。
2. 将助手消息操作图标移动到助手气泡下方左侧。
3. 更新静态资源版本参数，解决浏览器缓存导致样式未更新的问题。

## 新增内容

1. 新增消息行纵向布局规则（气泡在上、操作栏在下）。
2. 新增角色化操作栏对齐规则（用户右对齐、助手左对齐）。

## 主要变更

1. `.message-row` 从横向改为纵向布局，操作栏统一落在气泡下方。
2. 用户与助手消息的操作栏分别按企业常见审美做左右对齐。
3. `index.html` 静态资源版本号升级到 `v=0.5.1`，确保前端强制拉取新资源。

## 兼容性说明

1. 本次仅样式与静态资源引用变更，不涉及后端接口与数据结构。
2. 聊天功能行为不变，仅 UI 位置调整。

## 验证结果

1. 前端脚本语法检查通过：`node --check app/web/app.js`。

## 已实现

1. 消息操作栏位置微调（用户/助手均移动到气泡下方）  
   `app/web/styles.css`
2. 静态资源防缓存版本更新  
   `app/web/index.html`
